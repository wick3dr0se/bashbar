## Install
```bash
// download the repository
$ git clone https://github.com/wick3dr0se/bashbar; cd bashbar

// install bashbar to $PATH
# install bashbar /usr/local/bin
```

## Usage
```bash
bashbar <percent> [delay]
```

Legend
```
$  ...  executed as regular user
#  ...  executed as superuser
<>  ...  required argument
[]  ...  optional option
//  ...  comment
```
